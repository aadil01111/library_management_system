Library Management System (PHP + MySQL)

Project Overview

The Library Management System is a secure web-based application developed using PHP, MySQL, and Twig templating engine.
It supports role-based authentication, CRUD operations, advanced book search, captcha-protected login, and security best practices.

The system differentiates between Admin and User roles:

Admin can manage books (Create, Read, Update, Delete)
User can view and search books.

Features

Authentication & Authorization
Secure login system using password hashing
Role-based access control (admin / user)
Captcha protection on login page
Session-based authentication

User Management

Admin account (manual DB insertion)
User registration (Create Account)
Unique usernames enforced

Book Management (CRUD)

Add new books
Edit book details
Delete books
View all books

Advanced Search

Search books by:
Title
Author
Category (e.g., Sci-Fi)
Year (e.g., 2023)

Add to cart feature

User can add book to cart.
